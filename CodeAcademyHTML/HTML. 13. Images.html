Images

1. What is a local address and how does it differ from a web address?

Que es una direccion local y como se diferencia de una direccion web?

A local address refers to the path to an image, file, or resource that exists on your own computer. A web address on the other hand generally refers to the path to an image, file, or resource that exists externally, on a remote computer.

Una direccion local hace referencia a la ruta de una imagen, archivo o recurso que esta en la propia pc. Una direccion web hace referencia a la ruta hacia una imagen, archivo, o recurso que existe de forma externa en una pc remota

2. Can I link to images that are stored locally on my own computer?

Puedo vincular imagenes que estan almacenadas de forma local en mi propia pc?

Yes you can! In fact, developers often work locally before pushing their code to production. When dealing with assets like images, we will often want to include them in a folder which will also contain our HTML document. With a properly arranged directory or folder structure, we can create a path which points to our image and is relative to our HTML document. In this scenario, this relative path will be the value of the src attribute.

Si de hecho la mayoria de los desarrolladores trabajan de manera local antes de que su codigo se lleve a produccion. Elementos como imagenes que se quieren incluir se organizan por lo regular dentro de una carpeta. Organizada la estructura de directorios se puede crear una ruta que apunta a la imagen y sea relativa al documento HTML. En este caso, la ruta sera el valor en la etiqueta o src.

3. How can I store my own images online?

¿Cómo puedo almacenar mis propias imágenes en línea?

There are sites that allow you to upload your images to the web. Once your image is uploaded, you can then use its web address as the value for the element’s src attribute. More commonly, images and project assets are stored within a folder that also contains our HTML document. This allows developers to create paths to be used as src values that are relative to the HTML document. In other words, images and HTML files often exist on the same server or computer.

Hay sitios que permiten almacenar tus imagenes en la web. Una vez cargada la imagen, se puede usar su direccion web en el atributo src de la etiqueta img, por lo regular las imagenes y archivos HTML son almacenados dentro del mismo sitio.
